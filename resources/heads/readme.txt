==============================================================================
collection of resources for HRTFs from different dataset.
==============================================================================

IMPORTANT: 
original data are modified to comply with 3D-TuneIn requirements,
see https://3d-tune-in.eu/ for details.

Specifically the impulse responses inside the .sofa files are
modified to be zero-delay and the values of the delay is 
written into the specific variable inside the .sofa file.
